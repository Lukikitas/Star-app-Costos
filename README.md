# STAR Costos

App web para calcular costos y precios sugeridos de productos gastronómicos usando React + Firebase.

## Scripts
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run deploy`

## Firebase
- Authentication Email/Password
- Cloud Firestore
- Escucha en tiempo real con `onSnapshot`
- Timestamps con `serverTimestamp`

## Deploy GitHub Pages
- Configurar `base` en `vite.config.ts` según URL final.
- Incluye `404.html` para recarga de rutas.
- Incluye workflow `.github/workflows/deploy.yml`.
